CALCULATOR USER MANUAL

This calculator allows you to perform basic mathematical operations such as addition, subtraction, multiplication, and division. Additionally, it provides functions for calculating square roots, powers, factorials, and modulo. The calculator also includes a stack that allows you to store and use results. You can use the keyboard to enter numbers and mathematical operations.

OPERATIONS

+

Addition: Enter numbers and use the "+" button. (5+2 = 7)

-

Subtraction: Enter numbers and use the "-" button. (5-2 = 3)

×

Multiplication: Enter numbers and use the "×" button. (3×2 = 6)

÷

Division: Enter numbers and use the "÷" button. (8÷4 = 2)

FUNCTIONS

√

Root: Enter the index of the root and use the "√" button followed by the number that specifies the base of the root. (2√9 = 3)

^

Power: Enter the number you want to raise and use the "^" button followed by the exponent. (2³ = 2^3 = 8)

%

Modulo: Calculates the remainder after division. Use the "%" symbol. (10%3 = 1)

!

Factorial: Enter a number and use the "!" button. (3! = 6)

STACK

MS

Memory store: Stores the current result on the stack.

MR

Memory read: Uses the value at the top of the stack. If the stack is empty, the value 0 is used.

MC

Memory clear: Deletes the value at the top of the stack.

M+

Memmory add: Is used to add the value in the result to the value at the top of the stack.

M-

Memory substract: Is used to subtract the value in the result from the value at the top of the stack.

CLEARING

CE

Clear entry: Clears the last character.

OTHER

You can use parentheses to write more complex mathematical expressions. Use the "(" and ")" buttons to write parentheses.

In the upper left corner, there is a setting where you can switch between light and dark modes. You can also change the number of decimal places displayed in the settings.
